init install command :
"npm init -y"
first command install :
"npm install express cors mongoose dotenv"
second command install :
"npm install -g nodemon"
third command:
"npm install --save express body-parser"
forth command: used for generating unique ids
"npm install --save uuid"
fifth command: installing express validator
"npm install --save express-validator"
sixth command: installing axios (optional)
"npm install --save axios"
seventh command:
"npm install --save mongoose-unique-validator"
eighth command:
"npm install --save multer" -> we install this in order to read data in a none 
                            json format so that we can work with photos and other data.

commands for terminal:
"nodemon server" - used to run the server 

Note : Make sure to activate mongodb database after long time of no use!
